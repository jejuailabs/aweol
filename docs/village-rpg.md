# 우리 마을 조사대 — 설계

> 이 문서는 **왜 이렇게 만들었는가**를 적는다. 무엇을 만들었는가는 코드에 있다.
> 선생님이 고칠 곳은 `src/lib/local-sites.ts`, `src/lib/civic-places.ts`,
> `src/lib/village-rpg.ts` 세 표(表)다. 이 문서는 그 표를 어떤 생각으로
> 짰는지를 남긴다.

## 무엇을 만드는가

아이가 **마을 조사대의 견습 조사원**이 되어, 사람을 만나고 장소를 찾아가며
**조사 수첩**을 채운다. 수첩이 차면 등급이 오른다.

놀이처럼 보이지만, 채워지는 것은 **초등 사회과 그대로**다 —
우리 고장의 역사, 우리 고장의 공공기관, 우리 고장의 생활과 문화.

## 왜 심부름인가

읽으라고 두면 안 읽는다. **누가 시켜야 간다.**
그리고 **갔다가 돌아와야** 한다 — 갔다 온 이야기를 누군가에게 해야
그게 자기 것이 된다. 심부름은 그 왕복을 만들기 위한 장치다.

기관 안내를 그냥 벽에 붙여두었을 때와, 마을 담당이 "애월진성에 가서
읽어보고 와 줄래요?" 라고 했을 때는 아이가 하는 일이 다르다.

## 세 개의 축

조사 수첩은 세 칸으로 나뉜다. **한 축만 있으면 마을이 납작해진다.**

### 1. 시간축 — 우리 고장의 역사

유적을 조사할 때마다 **연표 카드**가 한 장씩 켜진다.
아이가 다 모으면 자기 손으로 만든 연표가 남는다.

| 때 | 곳 | 무엇을 배우나 |
|---|---|---|
| 7~8만 년 전 | 빌레못동굴 | 제주는 화산이 만든 섬 |
| 구석기 | 빌레못동굴 (황곰·순록 화석) | **제주가 육지와 이어져 있었다** |
| 초기철기~조선 | 곽지패총 | 한자리에 이천 년이 층층이 쌓인다 |
| 고려 1271~1273 | 항파두리 | 삼별초의 마지막 |
| 조선 1581 | 애월진성 | 바다를 지키던 성 |
| 조선~1950년대 | 구엄 돌염전 | 돌 위에서 소금을 얻던 살림 |
| 일제강점기 | 애월진성 → 애월공진학교 | 성돌이 뜯겨 나가고 학교가 섰다 |
| 오늘 | 애월초등학교 | 그 자리에서 우리가 배운다 |

**이 표가 이 게임의 심장이다.** 아이가 매일 뛰노는 운동장이 성터였고,
그 앞바다가 왜구를 살피던 자리였고, 그 산 밑 동굴에 곰이 살았다 —
이게 한 줄로 이어지면 고장이 달라 보인다.

### 2. 오늘축 — 우리 고장의 공공기관

기관을 조사하면 '동네 살림' 칸이 채워진다.
**중요한 것은 기관 하나하나가 아니라 그것들이 어떻게 얽혀 있는가다.**

그래서 심부름 하나가 **여러 기관을 거친다.** 이사 온 친구 이야기가 그렇다:

```
읍사무소(전입신고) → 학교(배정) → 우체국(주소가 있어야 편지가 온다)
   → 파출소(길·안전) → 도서관(회원증)
```

이게 진짜 행정이다. 하나씩 떼어 설명하면 외울 것이 다섯 개지만,
한 사건으로 꿰면 이야기가 하나다.

### 3. 살림축 — 우리 고장의 생활과 문화

돌염전, 밭담, 연화못, 금산공원, 들불축제.
**"왜 하필 여기서 그랬을까"** 를 묻는 칸이다.

- 왜 제주 사람은 **돌 위에서** 소금을 만들었나 → 모래밭이 없고 현무암이 있으니까
- 왜 밭마다 **돌담**을 둘렀나 → 바람이 세고, 돌을 치워야 밭이 되니까
- 왜 마을 한가운데 **연못**을 팠나 → 물이 땅으로 새는 섬이니까

답이 전부 **제주라는 땅**으로 돌아온다. 이게 지역 학습이다.

## 사실을 다루는 규칙

유적 이야기는 아이가 **자기 고장 역사로 외운다.** 그래서 선을 긋는다.

1. **출처를 화면에 그대로 보여준다.**
2. **자료마다 갈리는 숫자는 안 쓴다.**
   - 애월진성 둘레: 549척 / 255보 → 안 씀. 대신 *지금 남아 있는 북성 80여m*.
   - 구엄 돌염전 시작: 조선 명종 무렵 / 1702년 강제검 판관 → 안 씀.
     대신 *조선 시대부터 1950년대까지*.
3. 갈리는 연도는 **폭을 두고** 적는다 ("1920년대 초").
4. **모르면 안 쓴다.** 빈칸이 지어낸 문장보다 낫다.

### 4·3 은 아직 안 넣었다

애월읍에도 4·3 의 자취가 있고, 5~6학년 교육과정에 들어 있다.
그런데 **한 곳을 정확히 확인하지 못했고**, 이 주제는 어설프게 쓰면
안 쓰느니만 못하다. 선생님이 확인한 장소와 표현으로 넣는 것이 맞다고 봤다.
표에 자리(`minGrade`, `caution`)는 만들어 두었다.

## 거리 문제 — 마을 지도와 읍 지도를 나눈 이유

걸어다니는 마을 지도는 학교에서 **400m** 다. 그런데 조사할 곳들은 그보다 멀다.

| 곳 | 학교에서 |
|---|---|
| 애월진성 | 학교 안 |
| 애월읍사무소·우체국 | 걸어서 |
| 구엄 돌염전 | 동북동 3km 남짓 |
| 하가리 연화못 | 남동 2km 남짓 |
| 항파두리 | 동남동 4km 남짓 |
| 곽지패총 | 남서 2km 남짓 |
| 납읍 금산공원 | 남 3km 남짓 |
| 빌레못동굴 | 남 7km 남짓 |
| 새별오름 | 남남동 11km 남짓 |

그래서 **읍 지도**를 따로 둔다. 마을은 걸어 다니는 곳, 읍은 조사하러 가는 곳.

그리고 읍 지도는 **방위와 거리로** 그린다 — 측량 지도가 아니라 **모식도**다.
아이에게 필요한 것은 좌표가 아니라 "학교에서 서쪽으로 2km" 이고,
그게 마침 **방위를 배우는 일**이기도 하다.
정확한 자리는 선생님이 표에서 고칠 수 있다.

## 게임 장치

| 장치 | 왜 |
|---|---|
| **조사 수첩** | 지금 할 일이 안 보이면 아이는 마을에서 헤맨다 |
| **느낌표 ❗ / 물음표 ❓ / 메달 🏅** | 멀리서도 '저 사람에게 갈 일이 있다'가 보여야 한다 |
| **연표 카드** | 조사의 결과가 **남는 것**이어야 한다. 점수는 안 남는다 |
| **열쇠말** | 각 곳에서 낱말을 얻는다. 골든벨·OX 문제와 이어진다 |
| **등급** | 견습 → 3급 → 2급 → 1급 → 마을 박사 |
| **에피소드** | 심부름 여럿이 한 이야기. 낱개보다 이야기가 오래 남는다 |

**점수를 안 붙였다.** 순위표를 붙이면 빨리 넘기는 아이가 이긴다.
조사는 빨리 하는 일이 아니다.

## 선생님이 고치는 곳

| 파일 | 무엇 |
|---|---|
| `src/lib/local-sites.ts` | 유적·명소. 글·출처·방위·거리·연표 |
| `src/lib/civic-places.ts` | 기관. 사람·하는 일·이야기·놓인 물건 |
| `src/lib/village-rpg.ts` | 심부름과 에피소드. 누가 무엇을 시키고 무엇을 주나 |

세 표 모두 **한 줄 더 쓰면 하나 늘어난다.** 화면은 안 건드려도 된다.
검증 스크립트가 **막다른 심부름**(없는 곳으로 보내기)과
**끊긴 고리**(앞 심부름이 없는데 뒤가 뜨는 것)를 막는다.
